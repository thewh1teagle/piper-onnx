# piper-onnx

Use piper TTS with onnxruntime

## Features

- Fast
- Lightweight
- User friendly
- Support (Windows/Linux/macOS/Raspbian)

## Setup

```console
pip install -U piper-onnx
```

See [examples](examples) for usage.
